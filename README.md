# 🍎 FruitVision AI - Intelligent Fruit Quality Assessment System

An AI-powered fruit quality assessment system that analyzes fruit images to predict **freshness**, **ripeness**, and **edibility** using advanced computer vision and transformer-based deep learning models.

---

## 📖 Overview

FruitVision AI is designed to automate fruit quality inspection by analyzing images of fruits. The system detects fruits in an image and predicts:

- Freshness
- Ripeness
- Edibility

This project demonstrates the use of modern Vision Transformer (ViT) and DETR (Detection Transformer) models for intelligent image analysis.

---

## ✨ Features

- 🍎 Detects fruits from images
- 🌱 Predicts freshness
- 🍌 Estimates ripeness
- ✅ Determines edibility
- 🤖 AI-powered image analysis
- ⚡ Simple Python-based implementation

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Patlolla-Harshitha/FruitVision-AI-Intelligent-Fruit-Quality-Assessment-System.git
```

Move into the project folder

```bash
cd FruitVision-AI-Intelligent-Fruit-Quality-Assessment-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python freshness.py.py
```

If prompted, provide the path to a fruit image.

Example:

```
banana.jpeg
```
